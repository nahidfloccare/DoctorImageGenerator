========================================================================
SIMPLE FLUX WORKFLOW (No PuLID) - Works 100%
========================================================================

In ComfyUI, delete the PuLID nodes and create this simple workflow:

1. DualCLIPLoader
   - clip_name1: t5xxl_fp16.safetensors
   - clip_name2: clip_l.safetensors
   - type: flux

2. Load Diffusion Model
   - unet: flux1-dev-fp8.safetensors

3. Load VAE
   - vae_name: ae.safetensors

4. CLIP Text Encode (Prompt)
   - text: "professional medical doctor in white lab coat, warm smile, 
            stethoscope, modern hospital, professional photography, 
            cinematic lighting, 8k, photorealistic"
   - Connect clip from DualCLIPLoader

5. CLIP Text Encode (Negative) - Leave empty
   - text: ""
   - Connect clip from DualCLIPLoader

6. Empty Latent Image
   - width: 1024
   - height: 1024

7. FluxGuidance
   - conditioning: from positive prompt
   - guidance: 3.5

8. KSampler
   - model: from Load Diffusion Model
   - positive: from FluxGuidance
   - negative: from negative prompt
   - cfg: 1.0
   - steps: 20
   - sampler_name: euler
   - scheduler: simple

9. VAE Decode
   - samples: from KSampler
   - vae: from Load VAE

10. Save Image
    - images: from VAE Decode

This will generate perfect doctor images (generic face).
